# Telegram Translation Bot

A simple Telegram bot that translates text messages to Arabic using Google Translate.

## Installation

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`

## Usage

1. Set your bot token in `bot.py` if not set.
2. Run: `python bot.py`
3. Send text to the bot in Telegram for translation.

## Requirements

- Python 3.8+
- Telegram bot token

## Features

- Translates text to Arabic
- Fallback dictionary for common phrases
